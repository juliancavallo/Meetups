
# Meetups
Web API that allows creating meetups and know how many beers do you need


## Tech Stack

- **Backend**
  - .NET 5
  - EF Core 5.0
  - Swagger

- **Database**
  - SQL Server

- **Authentication**
  - JWT

- **Testing**
  - nUnit
  - Moq
  - Coverlet (code coverage)

- **Logging**
  - log4net

- **Language**
  - i18n
